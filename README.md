# NEXUS — Refined

This repository contains a refined and accessible version of the "NEXUS" demo landing page.

Files added on branch `refine/nexus-ui`:

- index.html — cleaned semantic markup and accessibility improvements
- css/style.css — refined, responsive stylesheet
- js/script.js — lightweight interactions (nav toggle, reveal, cursor glow)
- LICENSE.txt — original MIT license retained

How to preview locally:

1. Clone the repo: `git clone https://github.com/Bromtrader/brom-website.git`
2. Checkout the branch: `git checkout refine/nexus-ui`
3. Open `index.html` in your browser or serve with a static server.

You can download the repo ZIP from GitHub (Code → Download ZIP) after the branch is available.
